1. What is the difference between map, filter, find and forEach methods ?

Answer : map() will return a new array as per the conditions applied. forEach() will not return anything. forEach() returns undefined. filter() method will return an array of matching elements else will return an empty array if no matching happens.The find() method returns the value of the first element that passes a test.The find() method returns undefined if no elements are found.

2. What are the differences between localStorage, sessionStorage and Cookies ?

Answer :Local storage is useful for storing data that the user will need to access later, such as offline data. Session storage is a great way to improve the performance of your web applications. Cookies are a good choice for storing data that should not be persisted for a long time, such as session IDs.

3.What is Event Loop in javaScript?

Answer :The event loop is a single-threaded loop that monitors the call stack and checks if there is any work to be done in the task queue. If the call stack is empty and there are callback functions in the task queue, a function is dequeued and pushed onto the call stack to be executed.

quizTimer() and chooseQuiz() function

1.quizTimer() function

Answer :The quizTimer function manages a timer on a web page. If the dismiss parameter is true, it stops the timer. If not, it updates and displays a timer in the format "mm:ss sec" on the page, where "mm" is minutes, "ss" is seconds, and the text alternates color after 60 seconds. The timer counts upward, and you can dismiss it by passing dismiss as `true.

2.chooseQuiz() function

Answer :The chooseQuiz function is used to manage quiz. It checks if a quiz with a given index already exists in the answers array. If it exists, it updates the answer; if not, it adds a new answer object to the array. It also displays the answers. When the array reaches a length of 6, it enables a submit button and updates its styling to indicate readiness for submission.
